= Adding a value to an existing configuration snippet

If you want to add a value to an existing list, you can add it to an existing snippet.

.Prerequisite

* You installed {microshift-short}.
* You have access as a root user.

.Procedure

. For example, to add `hostZ` to an existing list of SANs, edit an existing snippet you have instead of creating a new one.
+
.Example starting `10-san.yaml` snippet
[source,yaml]
----
apiServer:
  subjectAltNames:
    - host1
    - host2
----
+
.Example edited `10-san.yaml` snippet
[source,yaml]
----
apiServer:
  subjectAltNames:
    - host1
    - host2
    - hostZ
----
+
.Example configuration result
[source,yaml]
----
apiServer:
  subjectAltNames:
    - host1
    - host2
    - hostZ
----
